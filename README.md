# 🎓 Student Performance Prediction System

An interactive Machine Learning web application built with **Python** and **Streamlit** that predicts student academic performance based on various academic parameters. The project also includes data analysis, visualization, and model evaluation to provide meaningful insights into student performance.

## 🌐 Live Demo

🚀 **Try the application here:**

**https://student-performance11.streamlit.app/**

---

## 📌 Features

- 📊 Exploratory Data Analysis (EDA)
- 📈 Interactive data visualizations
- 🤖 Student performance prediction using Machine Learning
- 📋 Dataset preview and summary statistics
- 📉 Feature importance analysis
- 🎯 Simple and user-friendly Streamlit interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/<YOUR_USERNAME>/<YOUR_REPOSITORY>.git
```

Move into the project directory

```bash
cd <YOUR_REPOSITORY>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
